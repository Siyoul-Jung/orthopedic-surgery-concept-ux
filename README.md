A high-fidelity, patient-centric website prototype for an orthopedic clinic, inspired by modern healthcare standards.

## 🚀 Project Overview
This prototype was developed to demonstrate a modern, accessible, and professional digital presence for a hand and upper extremity surgical center. It transitions from a tech-centric aesthetic to a warm, clinical 'Light Theme' that prioritizes patient comfort and practitioner expertise.

## ✨ Key Features
- **Clinical Light Theme**: A clean, healthcare-focused palette (White, Healing Teal, Clinical Navy).
- **Professional Imagery**: High-resolution, AI-generated medical photography to build immediate patient trust.
- **Anatomical SVG Icons**: Minimalist, professional iconography for specific upper extremity conditions.
- **Full Responsiveness**: Optimized for Desktop, Tablet, and Mobile devices with a functional hamburger menu.
- **Location Integration**: Dedicated St. Louis office section with direct Google Maps linking.
- **Glassmorphism UI**: Modern CSS effects for a premium, high-end look.

## 🛠️ Tech Stack
- **Languages**: HTML5, CSS3, Vanilla JavaScript
- **Typography**: Google Fonts (Outfit & Inter)
- **Deployment Ready**: Pure static files (HTML/CSS/JS) compatible with Vercel, Netlify, or GitHub Pages.

## 📁 Project Structure
- `index.html`: Main landing page and logic.
- `logo.jpg`: Official clinic branding.
- `surgeon.png`: Professional hero imagery.

## 💻 How to Run Locally
1. Clone the repository.
2. Open `index.html` in any modern web browser or use a live server extension.
3. (Optional) Run `npx serve` to simulate a production environment.
